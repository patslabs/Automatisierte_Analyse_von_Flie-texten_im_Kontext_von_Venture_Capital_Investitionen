README SELFMADE CLUSTERING APPLICATION

This application clusters a group of texts into different clusters that are created by the app itself.

***Prerequisites:***
Python 3
Python GUI of your choice (My choice: https://www.jetbrains.com/pycharm/)

Python Libraries (with the command to install them via the system command line(cmd)):
pip install pandas
pip install nltk
pip install gensim
pip install sklearn

At your first start run the following python code before:
import nltk
nltk.download('stopwords')


***Getting started:***
1. Check the prerequisites
2. Start your Python GUI and open the file "selfmade clustering.py"
3. Get the .csv file with the texts in the same folder as you have the application in.
4. Set the necessary variables, provided at the end of the application:
	"file" is the filename of the .csv file containing the texts.
5. Run the application
