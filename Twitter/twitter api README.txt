README TWITTER API APPLICATION

This application connects to the twitter api and pulls some tweet information.

***Prerequisites:***
Python 3
Python GUI of your choice (My choice: https://www.jetbrains.com/pycharm/)
Twitter Developer Account (https://developer.twitter.com)

Python Libraries (with the command to install them via the system command line(cmd)):
pip install tweepy


***Getting started:***
1. Check the prerequisites
2. Create a new twitter app in your twitter developer account or use the credentials of an existing one.
3. Start your Python GUI and open the file "twitter api.py"
4. Set the necessary variables, provided at the bottom of the application.
	"ACCESS_TOKEN", "ACCESS_SECRET", "CONSUMER_KEY", "CONSUMER_SECRET" are all provided by the twitter application and have the same name.
	"search" is the keyword for which you will search twitter. Use any search that you can do in the search bar of the typical twitter application.
	"filename" is the name of the file in which you want to save your search results. Add a .txt. It does not need to exist yet so you can just write anything with a .txt at the end.
	"amount_of_tweets" is the number of tweets you want to download, has to be a full number.
5. Run the file
6. Optional are some more parameters to get a more detailed documentation. For more information refeer to the documentation in the file.
