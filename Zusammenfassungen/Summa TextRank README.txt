README Summa TEXTRANK APPLICATION

This application summarizes a text by exporting its most important sentences and does this by using the suma python library.

***Prerequisites:***
Python 3
Python GUI of your choice (My choice: https://www.jetbrains.com/pycharm/)

Python Libraries (with the command to install them via the system command line(cmd)):
pip install summa


***Getting started:***
1. Check the prerequisites
2. Start your Python GUI and open the file "RSS Feed.py"
3. Set the necessary variables, provided at the bottom of the application to what fits to your PC.
	"directory" is the folder in which the .txt file is saved that contains the text for the summarization.
	"file_name" is the name of the file in the directory, specified by "directory", in which the text is that has to be summarized.
	"summary_file_name" the name of the file in which the summary fill be saved. Add a .txt at the end.
4. Run the application
