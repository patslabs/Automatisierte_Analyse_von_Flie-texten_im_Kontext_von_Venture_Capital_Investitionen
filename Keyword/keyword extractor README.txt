README KEYWORD EXTRACTOR APPLICATION

This application extracts keywords out of a text.

***Prerequisites:***
Python 3
Python GUI of your choice (My choice: https://www.jetbrains.com/pycharm/)

Python Libraries (with the command to install them via the system command line(cmd)):
pip install gensim


***Getting started:***
1. Check the prerequisites
2. Start your Python GUI and open the file "keyword extractor.py"
3. Set the necessary variables, provided at the bottom of the application to what fits to your PC.
	"export_file_name" is the file to which the results will be exported. Has to be a .csv file and has to be a new file.
	"filename" is the file containing the text from which the keywords are extracted.
	"working_directory" is the directory in which the file from "filename" is saved.
4. Run the application
